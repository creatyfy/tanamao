# Tá Na Mão - Delivery

O **Tá Na Mão** é um sistema completo de marketplace para delivery, contando com 4 perfis de acesso distintos e integrados em tempo real:
1. **Cliente:** Navega pelas lojas, monta o carrinho, aplica cupons e acompanha o pedido ao vivo no mapa.
2. **Loja Parceira:** Gerencia seu cardápio, cupons, recebe pedidos em tempo real e despacha para motoboys ou entrega própria.
3. **Motoboy:** Fica online para receber chamadas de entrega, aceita corridas e confirma entregas via código de segurança.
4. **Administrador Global:** Aprova cadastros de lojas e motoboys, gerencia taxas de comissão e tem a visão geral do faturamento da plataforma.

## 🚀 Fluxo de Teste Recomendado
1. **Acesse o Admin:** Faça login com as credenciais acima para acessar o Dashboard Global.
2. **Cadastre uma Loja:** Em uma aba anônima, crie uma conta de "Loja". Volte ao Admin e **aprove** o cadastro dela.
3. **Cadastre um Motoboy:** Crie uma conta de "Motoboy", aprove-o no Admin e deixe-o com o status **Online**.
4. **Faça um Pedido:** Crie uma conta de "Cliente", adicione um endereço e faça um pedido na loja criada.
5. **Acompanhe a Mágica:** Aceite o pedido no painel da Loja, marque como "Pronto" e veja a corrida tocar no app do Motoboy!
